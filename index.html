Companion Integration Setup

Bitfocus Companion can push live variable values (stream state, duration, etc.) to this dashboard automatically.

Step 1 — Enable Companion HTTP API

In Companion: Settings → Surfaces → enable Enable HTTP API (port 8000).

Step 2 — Create a Trigger

  1. Go to Triggers → click Add Trigger
  2. Set type to Time Interval → every 10 seconds
  3. Under Actions → Add Action → HTTP Request → Send a HTTP request

Step 3 — Configure the HTTP Action

MethodPOST
URLhttp://192.168.0.121:8080/companion
Content-Typeapplication/json
JSON StringifyUNCHECKED

Step 4 — Body (example for Web Presenter)

{
  "stream_state": "$(WebPresenter:stream_state)",
  "stream_duration": "$(WebPresenter:stream_duration)",
  "platform": "$(WebPresenter:platform)"
}

Replace WebPresenter with your Companion connection label. Add or remove lines for each variable you want.

Step 5 — Add Variables to an Equipment Card

  1. Enable Edit mode on the dashboard
  2. Click the edit (pencil) button on the equipment card
  3. Scroll to Companion Integration section
  4. Set Connection Label to match your Companion connection (e.g. WebPresenter)
  5. Add variables one per line in this format:
stream_state | Streaming State | badge
stream_duration | Duration | text
platform | Platform | text

Format: key | Label | badge or key | Label | text. The key must match what's in the POST body above.

Companion Button Setup

Companion buttons let you trigger any button on your Companion layout directly from an equipment card — shown as a red button.

Step 1 — Find your button location in Companion

In Companion, hover over the button you want to trigger. The location is shown as Page / Row / Column — for example, Page 1, Row 0, Column 6.

Step 2 — Add it to the equipment card

In the Companion Buttons field, add one button per line in this format:

Label | page | row | col

Example — a Web Presenter On Air button at page 1, row 0, col 6:

On Air | 1 | 0 | 6

You can add multiple buttons, one per line:

On Air | 1 | 0 | 6
Record | 1 | 0 | 7
Cut | 1 | 1 | 0

Requirements

  • Companion must be running and reachable on your network
  • Companion IP must be set in Setup → App Settings → Companion IP
  • Companion HTTP API must be enabled: Settings → Surfaces → Enable HTTP API
